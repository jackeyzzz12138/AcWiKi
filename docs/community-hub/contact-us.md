Ac-Wiki 项目组欢迎任何形式的交流与合作！如果你有任何关于 Ac-Wiki 的建议、意见或者想要参与到项目中来，请随时联系我们。

## 联系方式

**目前的唯二正式沟通渠道是 [GitHub](https://github.com/ac-wiki) 和 [Telegram 频道](https://t.me/s/AcWiki)：**

- **GitHub**: 你可以在 GitHub 上提交 [issue](https://github.com/Ac-Wiki/Ac-Wiki/issues) 来反馈问题或者提出建议，也可以通过 [pull request](https://github.com/Ac-Wiki/Ac-Wiki/pulls) 来贡献代码和内容，还可以在 [discussions](https://github.com/Ac-Wiki/Ac-Wiki/discussions) 中留下任何评论！
- **Telegram 频道**: 加入我们的 Telegram 频道，浏览不定时推送的实用内容，频道已启用 [私信](https://t.me/AcWiki/292) 功能，你可以直接在频道里与我们沟通交流

此外，我们也欢迎通过其他社交媒体平台进行交流：

- [Telegram 群组](https://t.me/AcFourm)：这是 Telegram 频道的关联群组，如果无法加入群组，请通过频道私信功能反馈问题
- [QQ 群](https://qm.qq.com/q/WJI3hgBcm4)：方便大部分网络不佳的同学讨论交流
- [Discord 服务器](https://discord.gg/SHH5z8hZA)：如果你喜欢 Discord，也可以加入我们的服务器进行交流
- [Matrix 房间](https://matrix.to/#/#AcFourm:matrix.org)：极客向的沟通平台，与 Telegram 群组的信息互通，如果你了解 Matrix，可以加入我们的房间进行交流
