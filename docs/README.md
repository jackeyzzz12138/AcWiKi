## ✨ 项目特色

<table>
  <tr>
    <td width="50%">

  <h3>🎓 全面覆盖</h3>
  从校园生活到学术技能，从学生优惠到网络安全，提供大学生全生命周期的知识支持。

  </td>
    <td width="50%">

  <h3>📖 开放共建</h3>
  所有内容开源开放，任何人都可以参与编辑和改进，共建属于大学生的知识库。

  </td>
  </tr>
  <tr>
    <td>

  <h3>🔧 现代技术栈</h3>
  基于 MkDocs Material 主题构建，支持深色模式、全文搜索、即时导航和博客功能。

  </td>
    <td>

  <h3>🛡 隐私优先</h3>
  无广告、无追踪、无商业目的，纯粹的公益知识共享项目。

  </td>
  </tr>
</table>

## 🤝 参与共建

Ac-Wiki 是一个由志愿者驱动的公益项目。每一份贡献——无论是修正一个错别字、补充一段知识点，还是提交一篇完整的文章——都弥足珍贵。

1. **Fork** 本仓库
2. **创建分支** 进行你的修改
3. **提交 Pull Request** 等待审核合并

详细指南请查阅 **[贡献文档](./community-hub/CONTRIBUTING.md)**

[![提交建议](https://img.shields.io/static/v1?label=&message=提交Issue建议&color=2196f3&style=for-the-badge&logo=github&logoColor=white)](https://github.com/Ac-Wiki/Ac-Wiki/issues/new)
